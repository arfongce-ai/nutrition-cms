# 몸가짐 실시간 칼로리 측정 사용설명서

## 1. 앱 열기

앱 주소에 접속하면 로그인 화면 없이 바로 카메라가 열립니다.

공개 주소: https://momgagym-nutrition.pages.dev

## 2. 촬영 준비

음식이나 식품 영양성분표를 화면 가운데 둥근 테두리 안에 맞춥니다.

가운데에 `음식 및 성분표를 촬영하세요` 안내가 깜빡이면 아직 음식 후보를 찾는 중입니다. 음식 후보가 잡히면 안내 문구가 자동으로 사라지고, 위쪽의 `실시간 자동 분석` 카드에 예상 결과가 표시됩니다.

## 3. 실시간 자동 분석 보기

촬영 전에도 위쪽 카드에서 예상 열량, 당류, 나트륨을 볼 수 있습니다.

현재 단계에서는 음식 사진을 먼저 분석하고, 영양성분표 숫자가 보이면 보조로 함께 반영합니다. 브라우저가 문자 인식을 지원하지 않는 기기에서는 촬영 후 직접 입력으로 보정할 수 있습니다.

## 4. 촬영하기

화면 아래 가운데의 빨간 촬영 버튼을 누릅니다.

하단 왼쪽 버튼은 오늘 기록, 하단 오른쪽 버튼은 설정입니다.

## 5. A4 리포트 확인

촬영 후 A4 카드가 열리면 앱이 음식 후보, 추정 중량, 열량, 탄수화물, 단백질, 지방, 당류, 나트륨을 계산합니다.

음식명이 틀리거나 양이 다르면 결과 화면에서 음식명과 먹은 양을 수정합니다. 수정하면 영양 계산도 다시 반영됩니다.

## 6. 한식처럼 여러 음식이 있을 때

밥, 국, 김치, 반찬이 함께 있는 한식은 한 번에 완벽히 맞추기 어렵습니다. 앱은 먼저 대표 음식 후보를 자동 추정하고, 사용자가 틀린 부분만 보정하는 방식으로 사용합니다.

예: 현미밥, 된장찌개, 배추김치, 고기반찬

국물, 김치, 소스가 많은 식사는 나트륨이 높게 잡힐 수 있으므로 리포트의 노랑/빨강 경고를 확인합니다.

## 7. 저장하기

결과가 맞으면 저장 버튼을 누릅니다. 저장된 결과는 오늘 기록에서 다시 볼 수 있습니다.

## 8. 정확도를 높이는 팁

- 음식은 가능하면 밝은 곳에서 촬영합니다.
- 음식 전체가 둥근 테두리 안에 들어오게 합니다.
- 포장식품은 영양성분표 글자가 선명하게 보이도록 따로 촬영합니다.
- 한식은 촬영 후 음식명과 양을 한 번 확인하면 정확도가 올라갑니다.
- 의학적 질환이나 도핑 관련 판단은 참고용이며, 실제 치료나 경기 출전 전에는 전문가와 공식 기관 확인이 필요합니다.
