# 몸가짐 실시간 칼로리 측정 사용설명서

앱 주소: https://momgagym-nutrition.pages.dev

## 사용 순서

1. **설정하기:** 오른쪽 아래 `설정`에서 신체 정보, 건강 상태, 운동 목적을 선택합니다.

2. **음식 촬영하기:** 음식 전체를 화면의 둥근 테두리 안에 맞추고 빨간 촬영 버튼을 누릅니다.

3. **총칼로리 확인하기:** 앱이 음식·반찬과 과일 개수를 인식해 총칼로리와 주요 영양소를 계산합니다.

4. **결과 보정하기:** 음식명, 중량 또는 개수가 다르면 결과 화면에서 수정합니다. 수정된 값으로 총칼로리가 즉시 다시 계산됩니다.

5. **저장하기:** 결과 화면의 `저장`을 누릅니다. 저장한 식단은 왼쪽 아래 `오늘 기록`에서 확인합니다.

## 촬영 팁

- 밝은 곳에서 음식 전체가 보이도록 촬영하세요.
- 여러 반찬은 서로 겹치지 않게 놓으면 가짓수 인식이 더 정확합니다.
- 포장식품은 제품명이나 영양성분표가 크게 보이도록 촬영하세요.
- 자동 계산 결과는 추정값이므로 음식명과 양을 한 번 확인하세요.
